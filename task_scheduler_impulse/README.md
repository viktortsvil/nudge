## Get started
1. Create .env file and add configurations
   ```bash
   FIREBASECONFIG = { FIREBASECONFIG };
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
    npx expo start
   ```

4a. Start the app in ios simulator

   ```bash
    i
   ```
4b. Start the app in web simulator 

   ```bash
    w
   ```
